# 🎨 Random Color Palette Generator

## 📌 Project Overview
This project is a simple and interactive **Random Color Palette Generator** built using **HTML, CSS, and JavaScript**.  
It generates random color palettes that users can use for design, UI inspiration, or development projects.

---

## 🚀 Features
- Generate random color palettes with one click
- Display multiple colors at once
- Show HEX color codes
- Clean and user-friendly UI
- Copy color codes easily (optional feature)

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript 

---

## ▶️ How to Run
1. Clone the repository
2. Run the file:
   random-color-palette-generator.js
